# VueJs Retro Console
Simple output console component for VueJs with classic limegreen color.

## How import?
```
import RetroConsole from "vuejs-retro-console";
```
### Live:
[Example](https://codesandbox.io/s/vue-retro-console-z76e4?file=/src/App.vue:0-209)

## Properties
### output
```
type: Array
```
### fullscreen
```
type: Boolean
```
### fontSize
```
type: Number
```
### style
```
type: Object

e.g.:
{
    height: '250px',
    width: '50%'
}
```
